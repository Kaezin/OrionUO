# Options tab 2

![options2](http://www.imageup.ru/img130/2766109/optionstab2.jpg)

1) Debug options - вывод отладочной информации в окно:

- No debug - Не выводить отладочную информацию;

- FPS only - Отображать только количество FPS;

- Debugging - Показывать отладочную информацию (Количество ФПС, масштаб, количество загруженных объектов и т.п.).

2) FPS rate - Желаемое количество ФПС (фактическое может отличаться от указанного здесь при больших нагрузках и/или при сладой видеокарте).

3) Reduce FPS when UO window is unactive - Урезать ФПС если окно неактивно.

4) Standard characters animation delay - Стандартная задержка между сменой кадров анимаций живых существ.

5) Standard items animation delay - Стандартная задержка между сменой кадров неодушевленных объектов.

6) Use scaling in game window - Использовать масштабирование игрового окна (тестовая версия).

7) Remove object's text with alpha-blending - Удалять текст объектов используя альфа-блендинг (плавное удаление текста).

8) Draw characters status mode - отображение статуса персонажа в мире:

- No draw - Не отображать;

- Above character (Text) - Отображать над персонажем (выводится текстом);

- Under character (Line) - Отображать под персонажем (рисуется гамп с линией ХП).

Draw characters status condition - условие для отображения статуса персонажа:

- Always - Всегда отображать;

- HP <> MaxHP - Отображать только если ХП не равно максимальному (больше или меньше);

- HP lower % - ХП меньше указанного количества % от максимального.

9) Change trees to stumps - Заменить деревья на пеньки;

10) Marking cave tiles - Разметка тайлов пещер;

11) Hide vegetation - Скрывать декоративную растительность;

12) No fields animation - Не анимировать филды;

13) Lock gumps moving - Возможность блокирования случайного закрытия или перемещения некоторых типов гампов. При включенной опции и зажатых Ctrl+Alt над доступными этой опции гампами появятся замочки. Зеленый - гамп не заблокирован, красный - заблокирован;

14) Chat need press 'Enter' to activate it - Для активации чата необходимо нажать Enter;

15) Hidden characters display mode - режим отображения персонажей в невидимости:

- Original - Без изменений;

- With alpha-blending, alpha - Используя альфа-блендинг, значение настраивается;

- With spectral color - Используя спектральный цвет;

- With special spectral color - Используя специальный спектральный цвет;

16) Change hidden characters mode only for your person - Использовать применение режима отображения невидимых персонажей только для себя;

17) Transparent spell icons, alpha - Использовать прозрачность для иконок заклинаний. Уровень настраивается;

18) Old style maximized statusbar - Расширенные статусбары в новых клиентах отображаются используя гампы старых;

19) Original party statusbar gump - Заменить статусбары от группы на обычные;

20) Colorize characters by state - Расскрашивать персонажей в цвета, согласно из состоянию (отравлен, парализован);

21) Change animated fields to tiles - Заменить филды на плитки;

22) Add paperdoll slots - Добавить слоты для мелких предметов в папердолл;

Scale images in slots - Масштабировать картинки в слотах папердолла (тестовая версия);

23) Remove statusbars without objects - Удалять статусбары, чьи объекты исчезли с экрана;

24) Show console entry mode under game window - Отображать режим ввода текста под игровым окном;

25) Draw aura under characters mode - Отображение ауры под персонажами:

- Never - Никогда не отображать;

- Only in war mode - Отображать только в режиме боя;

- Always - Всегда отображать.

26) Draw aura only if Ctral pressed - Отображать ауру только при зажатом Ctrl;

27) Screenshots format - Формат скринов:

- BMP;

- PNG;

- TIFF;

- JPEG (при использовании этого типа могут быть краши).

28) Remove or new draw objects displaying with use blending - Удалять/отображать объекты мира с использованием альфа-блендинга (плавное отображение/удаление);

29) Draw helmets on shroud in the world - Рисовать шлемы поверх шроудов на персонажах в мире;

30) Draw world map before all gumps - Рисовать карту мира перед всеми гампами и миром;

31) No draw roofs - Не рисовать крыши.

32) Highlight targt by type - Подсвечивать прицел в зависимости от его типа (корректно работает на RunUO серверах).

33) Display a world map immediately after entering the world - Отображать карту мира после входа в игру (процесс создания текстуры может быть довольно долгим).
